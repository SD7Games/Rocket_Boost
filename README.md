# Rocket Boost

Physics-based gameplay prototype built in Unity (C#), focused on movement, control, and collision handling.

Playable WebGL version:
👉 https://sd7games.itch.io/rocket-boost

---

## 🚀 Overview

A small prototype exploring physics-driven movement and fast gameplay loops.

---

## 🧩 Key Features

• Rigidbody-based movement (thrust, rotation, inertia)

• Collision handling and instant restart loop

• Simple obstacle-based level design

• Basic audio and visual feedback

---

## 🧠 Tech

Unity, C#, Rigidbody physics, Prefabs

---

## 🎯 Purpose

This project was used to explore physics-based gameplay and was extended to improve control and responsiveness.
