# Rocket Boost

Physics-based gameplay prototype built in Unity (C#), focused on responsive movement, collision handling, and stable control under physics constraints.

Playable WebGL version:  
👉 https://sd7games.itch.io/rocket-boost

---

## 🚀 Overview

This prototype focuses on designing responsive player controls using Unity physics, balancing thrust, rotation, and inertia for stable and predictable movement.

The main challenge was to achieve consistent control behavior while working within the limitations of Rigidbody-based physics.

---

## 🧩 Key Systems

### Physics-based Movement System

• Implemented thrust-driven movement using Rigidbody  
• Balanced acceleration, rotation, and inertia for responsive control  
• Ensured predictable behavior across different movement states  

---

### Collision Handling

• Designed instant restart loop for fast gameplay iteration  
• Implemented collision detection with clear failure states  
• Ensured consistent behavior across different obstacle types  

---

### Gameplay Loop

• Built fast retry loop to support continuous gameplay  
• Focused on responsiveness and minimal downtime between attempts  

---

### Player Feedback

• Added basic audio and visual feedback  
• Improved clarity of player actions and collision results  

---

## 📈 Results

• Stable and predictable physics-based movement  
• Improved responsiveness of player controls  
• Consistent collision handling across gameplay scenarios  
• Smooth and fast gameplay loop  

---

## 🛠 Tech

Unity, C#, Rigidbody Physics, Prefabs

## 🎯 Purpose

This project demonstrates practical handling of physics-based movement, control responsiveness, and gameplay loop design in Unity.
